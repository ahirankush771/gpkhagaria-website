## ✨ Features

### 🎨 Design & UI
- 🌙 **Dark / Light Mode** — One-click toggle with smooth transition
- 🌐 **Particle Network Background** — Live animated canvas
- 💎 **Glassmorphism Design** — Modern frosted glass card style
- 📱 **Fully Mobile Responsive** — Works on all screen sizes
- 🎞️ **Scroll Animations** — Smooth fade-up reveal effects

### 🔧 Functionality
- 🔍 **Built-in Search** — Press `Ctrl+K` or `/` to search anything
- 📋 **Live Notice Board** — Color-tagged notices (Exam / Event / Important)
- 📊 **Animated Counters** — Stats that count up on scroll
- 📝 **Student Inquiry Form** — Built-in contact form with validation
- 🔝 **Back to Top Button** — Smooth scroll to top
- 🍔 **Mobile Navigation** — Hamburger menu for small screens

### 📚 Sections
- 🏠 Hero — College intro with stats card
- 📢 Notice Board — Latest announcements + upcoming events
- 🏛️ Departments — All 6 departments with intake info
- ⚡ Facilities — 8 facilities with icons
- 🏆 Achievements — Animated milestone counters
- 📝 Admission — Step-by-step process + course list
- 🖼️ Gallery — Filterable photo grid
- 📞 Contact — Info cards + inquiry form
- 🦶 Footer — Links, socials, quick navigation

## 🚀 Deployment — GitHub Pages

### Step 1 — Fork or Clone
```bash
git clone https://github.com/ahirankush771/gpkhagaria-website.git
cd gpkhagaria-website
```

### Step 2 — Push to GitHub
```bash
git add .
git commit -m "🚀 Initial deployment"
git push origin main
```

### Step 3 — Enable GitHub Pages
```
GitHub Repo → Settings → Pages → Branch: main → Save
```

### Step 4 — Your site is live at:
```
https://ahirankush771.github.io/gpkhagaria-website
```

> ⏱️ Takes 1-2 minutes to go live after enabling Pages

## 🏆 About This Project

This website was built as a **CSE Department project** for **Government Polytechnic Khagaria** to demonstrate modern web development skills and showcase how an outdated government college website can be redesigned using just HTML, CSS, and JavaScript — with zero frameworks and zero cost hosting.

**Key Highlights:**
- Single file deployment — no npm, no webpack, no backend
- GitHub Pages hosting — 100% free
- Built in under 24 hours
- Original college data preserved accurately
